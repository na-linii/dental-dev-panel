## 1. Общая картина

### Архитектура платформы

```
Пациент (WhatsApp/Telegram/MAX)
    |
Wazzup Cloud / Telegram Bot API
    |
dental-chat (webhook handler + LangGraph runtime)
    |                          |
dental-core (agent logic)    python_gateway (CRM/IDENT)
    |
Langfuse (tracing, prompts) <--- dental-hub (управление)
    |                                |
Hub Frontend (React SPA)      Admin Panel Frontend
```

### Масштаб кодовой базы

| Проект | LOC | Язык | Фреймворк |
|--------|-----|------|-----------|
| dental-hub backend | ~1 364 | Python 3.12 | FastAPI |
| dental-hub frontend | ~7 039 | TypeScript | React 19, Vite 8 |
| dental-chat | ~19 600 | Python 3.11 | FastAPI, LangGraph |
| python_gateway | ~3 000 | Python | FastAPI |
| admin-panel-frontend | ~2 500 | TypeScript | React 18, Vite 6 |
| dental-e2e-tests | ~2 040 | Python | pytest, Selenium, Telethon |

**Итого: ~35 500 строк кода**

### Инфраструктура

- **9 Docker-контейнеров** (nginx, hub-api, langfuse-web/worker/postgres/clickhouse/redis/minio, ngrok)
- **CI/CD:** GitHub Actions -> SSH deploy на 158.160.85.19
- **Туннель:** ngrok (статический free domain)
- **БД:** PostgreSQL 17 (shared с Langfuse), ClickHouse, Redis

## 8. Cross-project Integration

### Протоколы взаимодействия

```
Hub -> Core: HTTP REST (health, config, chat, graph proxy)
Hub -> Langfuse: REST API (traces, datasets, prompts)
Core -> Langfuse: SDK (auto-tracing via LangGraph callbacks)
Chat -> Core: Embedded LangGraph (не HTTP!)
Hub -> Core: SSH Deploy (git clone + docker compose)
```

### API контракты

| Endpoint | Hub timeout | Статус |
|----------|-------------|--------|
| /health | 5s | OK |
| /config | 10s | OK |
| /chat | 30s | Может быть мало (Core до 60s) |
| /graph | 10s | OK |

### Версионные несоответствия

| Зависимость | dental-hub | dental-core | dental-chat |
|-------------|-----------|-------------|-------------|
| langfuse SDK | >=2.0.0 | **>=4.0.0** | >=2.0.0 |
| langgraph | — | **>=1.1.0** | **>=0.2.0** |
| Python | 3.11+ | **3.12+** | 3.11+ |
| FastAPI | >=0.104 | >=0.115 | >=0.104 |

### Проблемы интеграции

| Проблема | Severity |
|----------|----------|
| langfuse SDK 4.0 vs Server 3.x несовместимость | CRITICAL |
| langgraph 0.2 vs 1.1 расхождение | HIGH |
| Chat timeout 30s может быть мало | MEDIUM |
| Hub не валидирует ChatRequest перед proxy | MEDIUM |
| Непоследовательная обработка ошибок между проектами | MEDIUM |
