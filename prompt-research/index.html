  ... ещё 30+ записей в starsmile.yml

    ## Decision Rules (ортодонтия) — 4 правила
  1. Смена кап vs первые капы → спроси "уже на курсе?"
  2. Ревизия vs финал → по контексту жалобы vs. завершения
  3. Брекеты — установить/активация/снять
  4. Default → orthodontics.consultation
