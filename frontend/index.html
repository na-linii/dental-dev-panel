Как работает система

Dental Core

Инстанс клиники. 1 Docker = 1 клиника. Dental Router (LLM) → FAQ / Booking / Confirmation Agent → Tools → CRM Gateway (Google Sheets) + Chat Gateway (Telegram). PostgreSQL: Identity DB, Checkpointer, Knowledge Base, Action Queue. Модель gpt-5.4-mini + OpenRouter fallback. Промпты из Langfuse.

Dental Hub

Платформа управления. Langfuse (shared), Hub API (proxy, управление клиниками), ngrok (связь с серверами). 3D визуализатор, trace animation, конфигурация клиник. Архитектура строится динамически из /graph endpoint.

Разворачивание клиники

YAML конфиг + env vars → docker compose up → CLINIC_ID выбирает клинику → agent подключается к Langfuse через Hub. 3-4 клиники на один сервер Yandex. CI/CD: push в main → pytest → auto-deploy.
