Обновлено 29.03.2026

Как работает система

Dental Core — инстанс клиники. 1 Docker-контейнер = 1 клиника. Содержит: Dental Router → FAQ/Booking Agent → Tools. Два gateway: Chat Gateway (Telegram) и CRM Gateway (Google Sheets). Промпты из Langfuse, модель gpt-5.4-mini + OpenRouter fallback.

Dental Hub — платформа управления. Langfuse (shared для всех клиник), Hub API (управление клиниками, proxy), ngrok (связь с удалёнными серверами). Фронтенд: клиники, визуализатор, roadmap.

Разворачивание клиники: YAML конфиг + env vars → docker compose up → CLINIC_ID выбирает клинику → agent подключается к Langfuse через Hub. 3-4 клиники на один сервер Yandex.

В планах: Confirmation Agent (Валентин) — после него ядро полностью готово. Далее плагины: Telegram Business, IDENT adapter, MAX, Voice. Создание и настройка агентов через Dental Hub UI.
